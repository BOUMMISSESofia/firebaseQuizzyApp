L'application de Quiz EMSI est une plateforme interactive destinée aux étudiants de l'école EMSI pour tester et améliorer leurs connaissances en programmation. Développée avec Android en utilisant le langage Kotlin, cette application s'appuie sur Firebase pour la gestion des données et utilise des fichiers JSON pour l'importation des questions. Les étudiants peuvent participer à des quiz couvrant divers sujets de programmation, allant des concepts de base aux technologies web, en passant par l'histoire de la programmation. Grâce à une interface utilisateur intuitive, l'application offre une expérience d'apprentissage engageante et efficace, permettant aux étudiants de suivre leurs progrès et de renforcer leurs compétences en programmation.
